MAC软件推荐

2018-09-24
生活

以下推荐一些我使用的MAC软件，帮助大家绕过坑。主要分为：

  • 办公

  • 工具

  • 媒体&社交

  • 开发

如果要安装非APP Store的，可以参考链接设置。

1.办公

  • Microsoft Office：必备，虽然MAC的三件套也不错，但是架不住Office的受众太大。

  • PDF Expert：用过最好用的PDF阅读器，功能全，而且很快。

  • Snipaste：很好用的截图软件。

  • Fantastical 2：粗力度的日程管理，好用。

  • Typora：即显即用的markdown编辑器。

  • OmniGraffle：MAC上的visio。

  • OmniOutline：大纲软件，可以完成类似于思维导图的功能。

  • XMind：思维导图。

  • TexLive+TexStudio：我爱用LaTex！

  • Things3：TODO软件，比Fantastical2细粒度。

  • 邮件：自带或者用Office的outlook。

2.工具

  • Chrome

  • PopClip：生产力工具，推荐。

  • Alfred3：神器。配合上一些workflow，把很多软件都省了。

  • Tuxera Disk Manager：读取移动硬盘等。

  • Dr. Unarchiver：界面类似于Win上面的解压软件。

  • Dr. Cleaner：免费的垃圾清扫软件。

  • Noizio：白噪声软件。

  • 印象笔记：有道、为知笔记亦可

  • Amphetamine：安非他命，让你的mac步自动息屏。

3.媒体&社交

  • IINA：播放器。

  • PhotoShop

  • 微信

  • QQ

  • sketch：mac上的矢量图制作软件

4.开发

  • brew：MAC下的命令行软件管理器。

  • iTerm2：好用的命令行。iTerm2+zsh=excellent！

  • vscode

  • Anaconda

  • R+R Studio

  • PyCharm Professional：可以远程编程。不得不说，jetbrains的ide都很给力。

  • Xcode

  • Dash

  • Transmit：FTP工具

  • Matlab

  • neovim+spaceVim

  • 字体：powerline, source code pro, fira code

  • 命令行工具：tmux, htop, ag, tldr, fzf, thefuck, mtr, cloc, jq

  • oh-my-zsh扩展：zsh-autosuggestions, zsh-syntax-hightlighting
